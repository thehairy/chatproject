
Chat-Raum mit Python3
Autoren: Sören Stabenow & Alan Kulinski

Als Vorlage wurde das Python2 Script von 
https://www.geeksforgeeks.org/simple-chat-room-using-python/
verwendet.

Dieses wurde dann auf Python3 angepasst und den Anforderungen entsprechend bearbeitet.

How to use

Server:
Das Script "server.py" wie folgt ausführen: "python3 server.py"
Standardmäßig wird die IP 127.0.0.1 und der Port 4242 verwendet.
IP und Port können manuell angegeben werden: "python3 server.py ip port"

Client:
Das Script "client.py" wie folgt ausführen: "python3 client.py"
Standardmäßig wird die IP 127.0.0.1 und der Port 4242 verwendet.
IP und Port können manuell angegeben werden: "python3 client.py ip port"

Folgende Befehle können verwendet werden:
/quit - Trennt die Verbindung
/nickname name - Ändert den Namen
